## 🏀 슬램톡 서비스 소개

> 주변에 있는 농구장을 검색할 수 있고, 해당 농구장을 이용하는 사람들과 채팅을 통해 소통할 수 있으며 농구 메이트 찾기, 팀 매칭, 대관 양도, 중고 거래 등의 기능을 이용할 수 있는 플랫폼입니다. <br>
>
> 팀 구성: 프론트엔드 3명, 백엔드 4명<br>
> 개발 기간: 24/01/11 ~ 24/2/22(프로젝트 발표) 이후 유지보수 중<br>
>
